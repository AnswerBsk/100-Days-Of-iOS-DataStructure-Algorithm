## 这是什么东西
很多技术越往底层大致皆相似，皆是数据结构与算法的缩影。这就是为什么一线互联网公司面试时候特别注重这方面的考察，国外公司甚至很多轮面试都是这方面考察。每篇文章除了有文字描述，有的还有图片以及动画方便大家理解，还会有源代码Demo，尽量用通俗易懂的方式让大家明白。

## 怎么做呢
每天会更新一篇数据结构与算法方面的东西，大家也可以跟着做，监督自己进步。
